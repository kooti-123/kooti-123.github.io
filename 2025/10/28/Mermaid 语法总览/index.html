Mermaid 语法总览

发表于 2025-10-28 分类于 技术 ， 语法/格式 本文字数： 1.7k 阅读时长 ≈ 3 分钟
Mermaid 是一个可以在 Markdown 中绘制图表和流程图的轻量级工具。Typora 支持 Mermaid，只需在 Markdown 中直接嵌入即可

1. 流程图 (Flowchart)

flowchart TD
    A[开始] --> B{判断条件}
    B -->|是| C[执行操作1]
    B -->|否| D[执行操作2]
    C --> E[结束]
    D --> E

2. 时序图 (Sequence Diagram)

sequenceDiagram
    participant Alice
    participant Bob
    Alice->>Bob: 你好，Bob
    Bob-->>Alice: 你好，Alice
    Alice->>Bob: 你在做什么？

3. 甘特图 (Gantt Chart)

gantt
    title 项目进度
    dateFormat  YYYY-MM-DD
    section 设计
    需求分析       :a1, 2025-10-01, 7d
    原型设计       :after a1, 5d
    section 开发
    前端开发       :2025-10-10, 10d
    后端开发       :2025-10-12, 12d

4. 类图 (Class Diagram)

classDiagram
    class Animal {
        +String name
        +eat()
    }
    class Dog {
        +bark()
    }
    Animal <|-- Dog

5. 状态图 (State Diagram)

stateDiagram-v2
    [*] --> 空闲
    空闲 --> 工作
    工作 --> 空闲
    工作 --> 停止
    停止 --> [*]

6. 饼图 (Pie Chart)

pie
    title 网站流量来源
    "直接访问" : 45
    "搜索引擎" : 30
    "社交媒体" : 25

7. 用户旅程图 (Journey Diagram)

journey
    title 用户注册流程
    section 注册
      访问页面: 5: 用户
      填写表单: 4: 用户
      提交信息: 3: 用户
    section 验证
      邮箱验证: 2: 用户
      完成注册: 5: 用户

8. ER 图 (Entity Relationship Diagram)

erDiagram
    CUSTOMER ||--o{ ORDER : places
    ORDER ||--|{ LINE_ITEM : contains
    CUSTOMER }|..|{ DELIVERY_ADDRESS : uses

9. Mermaid 小技巧

  • 使用 style 自定义节点颜色和样式
  • 箭头类型可选 -->、---、-.-> 等
  • 节点形状：[] 矩形，() 椭圆，{} 决策
  • 可以嵌套 subgraph

提示

  1. 在 Typora 中确保已启用 Mermaid 渲染（通常在 偏好设置 -> Markdown -> 渲染 中）。
  2. 所有 Mermaid 图都以 mermaid 包裹即可直接渲染。
  3. 可在 Typora 中直接编辑文本和 Mermaid 代码，实时预览效果。
